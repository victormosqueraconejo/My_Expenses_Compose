# My Expenses Compose

Base de arquitectura con Compose + MVVM + Room.

## Estructura
- `app/src/main/java/com/victor/myexpensescompose/data`: Room y repositorios.
- `app/src/main/java/com/victor/myexpensescompose/domain`: modelos y contratos.
- `app/src/main/java/com/victor/myexpensescompose/presentation`: UI y ViewModels.

## Ejecutar pruebas unitarias
```sh
./gradlew test
```
